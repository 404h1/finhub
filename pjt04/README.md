# PJT 04 · 금융 자산 토론 게시판

## 실행
```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

## 요구사항
| 번호 | 내용 | 구분 |
|------|------|------|
| F401 | 자산 목록 출력 (assets.json) | 필수 |
| F402 | Post 모델 설계 | 필수 |
| F403 | 자산별 게시판 | 필수 |
| F404~F407 | 게시글 CRUD | 필수 |
| F408 | 404 예외 처리 | 필수 |
| F411 | 환경변수 설정 | 심화 |
| F412 | LLM 부적절 표현 필터링 | 심화 |
